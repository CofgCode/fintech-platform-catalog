# FinTech Platform Catalog

This repository contains the Backstage catalog entities for the FinTech Platform. It defines the organizational structure (users and groups), systems, and services.

## Organization
- `org/groups.yaml`: Defines the teams (platform-team, banca-digital-team, creditos-team).
- `org/users.yaml`: Defines the users and their team memberships.

## Systems
- `systems/`: Contains the system definitions.

## Services
- `services/`: Contains the service components and their APIs.
